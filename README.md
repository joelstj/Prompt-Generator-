# ⚡ AI Prompt Generator — Blockchain & DeFi Dev Edition

A professional dark-themed web application that helps developers build structured, optimized AI prompts for blockchain, DeFi, and bot-building projects. Fill in six purpose-built fields, choose from 12 expert templates, and generate a ready-to-use prompt in seconds.

---

## ✨ Features

- **6-Field Structured Builder** — Main Goal, Output Format, Rules to Follow, Output Goal, Correctness & Verification, How to Act
- **12 Blockchain/DeFi Templates** — Solidity contracts, DeFi bots, arbitrage, MEV, NFTs, cross-chain, and more
- **Smart Blockchain Context** — Auto-detects blockchain keywords and injects expert system context
- **One-Click Copy** — Copy generated prompts to clipboard instantly
- **Template Library** — Filterable card-based library with category badges
- **Dark Professional UI** — GitHub-inspired dark theme with responsive layout
- **REST API** — Simple JSON API for programmatic use

---

## 🛠 Prerequisites

- **Python 3.9+**
- **pip** (bundled with Python)
- A modern web browser

## 🚀 Quick Start (Recommended)

```bash
# 1. Clone the repository
git clone https://github.com/your-username/Prompt-Generator-.git
cd Prompt-Generator-

# 2. Run the installer
chmod +x install.sh
./install.sh

# 3. Start the app
./run.sh

# 4. Open in browser
# http://localhost:5000
```

## 🔧 Manual Installation

```bash
# Create and activate virtual environment
python3 -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the Flask app
python app.py
```

Then open **http://localhost:5000** in your browser.

### Debug mode

```bash
FLASK_DEBUG=true python app.py
```

## 📖 Usage

1. **Select a template** from the dropdown or the Template Library cards below the editor — all six fields are pre-filled automatically.
2. **Edit the fields** to customise the prompt for your specific use case.
3. Click **⚡ Generate Optimized Prompt** — the structured prompt appears in the right panel.
4. Click **📋 Copy to Clipboard** and paste into your AI assistant (ChatGPT, Claude, Gemini, etc.).

## 🔌 API Endpoints

### `GET /api/templates`
Returns all prompt templates as a JSON array.

```json
[
  {
    "id": "solidity-dev",
    "name": "Solidity Smart Contract Developer",
    "category": "Smart Contract",
    "description": "...",
    "main_goal": "...",
    "output_format": "...",
    "rules": "...",
    "output_goal": "...",
    "correctness": "...",
    "how_to_act": "..."
  }
]
```

### `POST /api/generate`
Generate an optimized prompt from the six input fields.

**Request body:**
```json
{
  "main_goal":     "string",
  "output_format": "string",
  "rules":         "string",
  "output_goal":   "string",
  "correctness":   "string",
  "how_to_act":    "string"
}
```

**Response:**
```json
{
  "prompt":     "## MAIN GOAL\n...",
  "char_count": 1234
}
```
